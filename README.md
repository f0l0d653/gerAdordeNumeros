# Descrição

Aplicação Web voltada à geração de números aleatórios com base em parâmetros dinâmicos definidos pelo usuário. O projeto foca no fortalecimento da lógica de programação e na manipulação eficiente do DOM, garantindo que as regras de intervalo e quantidade sejam processadas em tempo real.

# Tecnologias usadas
- <kbd> HTML5 </kbd>
- <kbd> CSS3 </kbd>
- <kbd> JavaScrip </kbd>
